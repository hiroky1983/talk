description: セッション開始時にメモリバンクから記憶を想起する
---

メモリバンクから現在のプロジェクト状況と過去の知見を読み込みます。

1. `docs/ai/memory/index.md` を読み、現在の優先事項を確認してください。
2. `docs/ai/memory/architecture.md` を読み、設計方針を再確認してください。
3. `docs/ai/memory/gotchas.md` を読み、ハマりやすいポイントを把握してください。
4. 読み込んだ内容に基づき、今日の作業における「成功の定義」と「回避すべきリスク」を3行で要約してください。
