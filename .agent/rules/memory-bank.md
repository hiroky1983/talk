# AI Memory Bank Rules

## エージェントの行動指針
1. **想起 (Recollection)**: 作業を開始する前、または新しいタスクに取り組む前に、必ず `docs/ai/memory/index.md` を読み、現在のコンテキストと過去の決定事項を把握すること。
2. **記録 (Recording)**: 
   - 重要なアーキテクチャ上の決定を行った場合、即座に `docs/ai/memory/decisions.md` に ADR として記録すること。
   - 解決に時間がかかったバグや、特定の環境でのみ発生する罠を発見した場合、`docs/ai/memory/gotchas.md` に追記すること。
3. **蒸留 (Distillation)**: セッションの終了時に、そのセッションで得た知見を整理し、メモリバンクの各ファイルへ適切に反映させること。

## 書き方のルール
- **簡潔さ**: 箇条書きを多用し、AI が次回以降に高速にスキャンできるように記述する。
- **日付**: 決定事項や罠には日付を付与する。
- **リンク**: ファイル間の関連がある場合は、Markdown リンクを使用して参照可能性を高める。
